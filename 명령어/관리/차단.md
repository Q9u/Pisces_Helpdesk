# 차단
대상을 차단합니다.

## 사용 방법
아래 문장을 입력하세요.
```
,,차단 [대상] (사유)
```

> ### 매개변수
> * 대상 (필수)
>   > * `사용자 언급` 또는 `사용자 ID`입니다.
> * 사유
>   > * 차단 사유를 입력하세요. 입력하지 않은 경우 기본값으로 지정합니다.

## 사용 예시
* 일반 사용
> `,,차단 @예시#0000`

* 사유 지정
> `,,차단 @예시#0000 서버 규칙 위반`

## 별칭
아래 이름으로도 이 명령을 호출할 수 있습니다.

* '밴'
* 'ban'
