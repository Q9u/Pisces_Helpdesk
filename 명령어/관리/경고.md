# 경고
대상의 경고를 조정합니다.

## 사용 방법

### 경고 추가
아래 문장을 입력하세요.
```
,,경고 추가 [대상] (사유)
```

### 경고 차감
아래 문장을 입력하세요.
```
,,경고 차감 [대상] (사유)
```

### 경고 초기화
아래 문장을 입력하세요.
```
,,경고 초기화 [대상] (사유)
```

### 경고 보기
아래 문장을 입력하세요.
```
,,경고 보기 [대상]
```

> ### 공통 매개변수
> * 대상 (필수)
>   > * `사용자 언급` 또는 `사용자 ID`입니다.
> * 사유
>   > * 경고 추가 또는 차감, 초기화 사유를 입력하세요.

## 사용 예시
* 경고 추가
> `,,경고 추가 @예시#0000`

* 경고 차감
> `,,경고 차감 @예시#0000`

* 경고 초기화
> `,,경고 초기화 @예시#0000`

* 사유 지정
> `,,경고 추가 @예시#0000 경고 추가 사유`
> `,,경고 차감 @예시#0000 경고 차감 사유`
> `,,경고 초기화 @예시#0000 경고 초기화 사유`
