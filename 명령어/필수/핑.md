# 핑
봇의 응답 시간을 계산합니다.

## 사용 방법
아래 문장을 입력하세요.
```
,,핑 (옵션)
```

> ### 매개변수
> * 옵션
>   > * '시간'
>   >   > 봇의 응답 시간을 출력합니다.
>   > * 'API'
>   >   > Discord API의 응답 시간을 출력합니다.

## 사용 예시
* 일반 사용
> `,,핑`

* 응답 시간 출력
> `,,핑 시간`

* Discord API 응답 시간 출력
> `,,프사 API`
