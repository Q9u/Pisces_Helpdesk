# 봇
봇의 정보를 봅니다.

## 사용 방법
아래 문장을 입력하세요.
```
,,봇 (옵션)
```

> ### 매개변수
> * 옵션
>   > * '버전'
>   >   > 봇의 버전에 대한 정보만 불러옵니다.
>   > * '초대'
>   >   > 봇의 초대 링크를 불러옵니다.

## 사용 예시
* 일반 사용
> `,,봇`

* 버전 정보 보비
> `,, 봇 버전`

* 봇 초대 링크 불러오기
> `,, 봇 초대`

## 별칭
아래 이름으로도 이 명령을 호출할 수 있습니다.

* 'bot'
* 'botinfo'
